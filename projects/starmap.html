Created as a part of ASTRAL.INSTITUTE 2025 at Swimbourne.

The starmap project collected the data of 600k+ stars from the GAIA and SIMBAD astronomy databases.

Stars were filtered, visible stars (stars with apparent magnitudes of less than 8), were compiled.
More than 16k stars remained. This catalouge was then amended with more than 100 very bright stars from outside the extracted GAIA catalouge.
(the GAIA satelite is very sensitive, it does not observe known bright stars to preserve onboard instruments)

Using the Python PIL module, these stars were drawn on a large png canvas.

Constellation data came from the IAU (International Astronomical Union).
This currently uses conventional Western skylore, with current work to include the Boorong constalations of Australian Indigenous skylore of the Boorong clan.
The specific JSON file may be found HERE.
Using PIL, these were also drawn on the png canvas.
Special code was used to ensure seamless connection across borders of the image file.

The specific renderer has modifyable configuration that may be adjusted.
Currently it is configured to create a high quality, 2 by 1 image, perfect to be used as a skybox within the Unity engine, like we did for the Planet Hunter project.


Note: skymap is aligned with poles of the earth (uses RA DEC), it is not aligned with the ecliptic plane
