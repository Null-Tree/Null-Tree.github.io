Pixel art background

The resources I used are as follows:
Pixel art editor: Pixilart
Adobe Photoshop

As this is a tessalating pixel art, it is important to give yourself guidelines to see where you need parts of the art to join up

Once you have these guidelines up, you can start making simple pixel arts of buildings and layering them. Keeping in mind to match the buildings with the guidelines if they ever intersect with the boundaries of the page.
Once you have your buildings, you can create the lit up windows.
Make sure to create these lights on a different layer. This will save alot of time later on.
Things to keep in mind with lights:
  • Try to keep them evenly distributed, so that density patterns dont form when the art is tessalated.
  • Avoid putting in too many either, as if theres two many, when we add glow effects later they may end up forming patterns and lines when the glow effects overlap. As humans are really sensitive to patterns, it will become really clear if the image is repeated.
If these were not considered, the image may end up looking like this instead, where the repeating pattern is very clear:
Now you have finished making the lights, make sure to export the base layer and layer with lights as seperate files. And open them on photoshop.
You will want to scale up these files, as otherwise the light effects will be pixelated. If you dont want a higher definition glow effect, and prefer a more pixelated style, you may skip this.
To scale up, go into image>image size, and adjust dimensions to your desire.
For pixel art make sure to select "Nearest Neighbour (hard edges)" mode, which maintains the crisp edges. Otherwise your pixel art will end up blurry


Now create a new project in the same dimensions you scaled up to. And move the base and light pictures onto this project in diffrent layers
There are some things to keep in mind with moving the light layer.
Because the image file of the lights is mostly empty except for the lights themselves, you cannot do Crtl-A then copy, because photoshop cannot handle empty borders, and will automatically crop your selection
So aligning may become tricky.
Personally I recommend putting placeholder pixels at a pair of opposite corners of the image, and then copying it across.
After both base layer and lights layer are on. You are ready to add the glow effects.

There are many ways to do this, but these are the two guides I followed:
Glow effect guide by Benny Glow guide by Spoon graphics
Try avoid Crtl-A then copying layers to duplicate them, as it pastes the new layer in diffrent position. Right click layer and duplicate instead.

As you are trying to make a tessalating tile. having glow effects that do not line up may also be ploblemmatic. This does not matter too much for the glow effect that uses gaussian blur, unless you have lights right up agaist the border. For the glow effect using color lookup and a low flow brush, I recommend the below. Split the art into two sections using guides set at quarters.
And apply the glow effects seperately.
You would want to create two diffrent copies of: base and light layer, and the color look up layer.
For the inner section, go ahead and apply the glow effects
But prior to applying the same glow effects to the outer lights, you will want to 'transform' the image as below. To make sure each light effect will line up even when tessalated. You will want to translate each quadrant to its opposite quadrant. Eg Top Right -> Bottom left Below are images before and after transformation. Square is there for reference.

Now when you add the glow effects for these lights, the glow effect will now line up across borders. Once you add them you will want to do the transformation again this time to the color lookup layer, to revert it to line up with the original image.
If nothing has gone astray, your tessalating tile will have been completed.
