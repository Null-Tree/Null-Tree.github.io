A solar system simulation.

The project was a simulation of the solar system; with planets, orbits, planet rings etc.
Camera is positioned on earth, following earth orbit, but angled so it never shows the sun.
Planets are clickable, once found will enter into a more detailed view scene.

This project was originally intended for a very large LED screen at the Swimbourne campus.
As such planets are scaled for a ?k screen, some planets maybe too small to see on smaller screens.
The original screen does support 3D stereo visuals, thus there are plans to implement compatability for 3D screens.

We used the Unity game engine for this project.
Ackowedgements of contributers may be found in the credits of the project.
The project uses the Astral starmap exports as the skybox.
However the project implements the ecliptic plane flat on the X Z axis, thus the starmap is misaligned by around 23 degrees.
Need to either tilt the Unity skybox or change render methord of original starmap.
